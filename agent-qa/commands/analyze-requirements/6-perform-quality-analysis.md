# PHASE 6: Perform Quality Analysis

Perform requirement completeness and quality analysis.

## Core Responsibilities

1. **Check Completeness**: Analyze requirement completeness
2. **Calculate Completeness Score**: Generate completeness scores
3. **Perform Quality Scoring**: Calculate quality scores
4. **Generate Recommendations**: Provide improvement recommendations

## Workflow

### Step 1: Check Requirement Completeness

For each requirement:
- Check for required fields (summary, description, acceptance criteria)
- Identify missing fields
- Calculate completeness percentage

### Step 2: Calculate Completeness Score

For each requirement:
- Score: (present fields / total required fields) * 100
- Level: High (80-100%), Medium (50-79%), Low (<50%)

### Step 3: Perform Quality Scoring

For each requirement:
- Analyze clarity, completeness, acceptance criteria quality
- Calculate quality score (0-100)
- Level: Excellent (90-100), Good (70-89), Fair (50-69), Poor (<50)

### Step 4: Generate Recommendations

For each requirement:
- Identify areas for improvement
- Generate actionable recommendations

## Important Constraints

- Use consistent scoring methodology
- Provide actionable recommendations
- Store scores in requirement structures
