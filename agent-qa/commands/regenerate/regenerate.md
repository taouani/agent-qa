# Regenerate Command

You are helping to regenerate deliverables that were affected by requirement changes for agent-qa.

This command detects changes in source requirements since the last generation, identifies which deliverables are affected, and regenerates only those deliverables.

**Input**: User selection of existing output folder
**Output**: Updated deliverables in the same output folder
