# Publish to Confluence Command

You are helping to convert Agent-QA deliverables to Confluence storage format and optionally publish them to Confluence.

This command finds existing deliverable output folders, converts selected deliverables to Confluence storage format (XHTML), and optionally publishes them via the Atlassian MCP server.

**Input**: User selection of output folder and deliverables to convert/publish
**Output**: `.confluence.html` files alongside original markdown, optional Confluence page creation
